AI Tool Reference

ChatGPT

Versatile tool for brainstorming, content generation, and analysis

Best For:

  • General brainstorming
  • Content generation
  • Structured analysis
  • Persona creation

Prompt Tips:

  • Use specific role definitions
  • Provide detailed context
  • Request structured outputs
  • Iterate based on results

Claude with Artifacts

Superior analytical capabilities with interactive prototype creation

Best For:

  • Rapid prototyping
  • Complex reasoning
  • Interactive elements
  • Collaborative iteration

Prompt Tips:

  • Describe UI elements clearly
  • Specify interaction patterns
  • Request iterative improvements
  • Use for multi-step reasoning

Perplexity AI

Research-focused with real-time web search and citations

Best For:

  • Competitive analysis
  • Trend research
  • Market insights
  • Fact verification

Prompt Tips:

  • Ask for recent information
  • Request source citations
  • Use for factual verification
  • Combine multiple research angles
